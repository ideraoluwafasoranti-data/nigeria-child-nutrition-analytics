# An Analysis of Infant Dietary Diversity and Childhood Wasting in Nigeria

## Project Overview
This independent research project uses microdata from the 2024 Nigeria Demographic and Health Survey to look at what causes child malnutrition across different states in Nigeria. 

With my background in Nutrition and Dietetics, my goal is to shift from looking at nutrition on an individual level to understanding it at a population level. Using R programming, I wanted to analyze how infant feeding practices relate to childhood wasting. Specifically, this project measures the percentage of babies aged 6 to 23 months who get a diverse diet versus those who are given sugary junk foods, and looks at how those habits correlate with wasting rates across different regions of Nigeria.

## Technical Tools Used
Language: R Programming
Software: RStudio
Key Packages: `tidyverse` (for cleaning data and plotting graphs), `haven` (for opening the DHS data files)
Statistical Methods: Descriptive statistics, Cross-tabulation, Multiple Linear Regression

## Project Files
`nutrition_analysis.R`: The R script used to clean the data and run the regression model.
`nutrition_chart.png`: The scatter plot graph generated by the code.
